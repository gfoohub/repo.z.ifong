◎ 檔案傳輸 -- ftp 的使用 ◎

○ 第一節 ftp 常用指令介紹 ○

  □ FTP (File Transfer Protocol) 是 Internet 上標準的檔案傳輸協定，ftp 則
     是使用此傳輸協定，應用在網路檔案傳輸的程式。ftp 的格式如下：
        ftp [IP or domain_name]
     其參數的意義和 telnet 一樣，例：ftp ftp.ntu.edu.tw
  □ 進入 ftp 之後，如果與 remote host 連上了，它將會自動執行登入的動作，
     詢問你 username 和密碼，確認完畢後，就可看到提示符號 "ftp>" ，開始進
     行檔案傳輸。
  □ 以下介紹 ftp 的指令：（註：local host 是使用者下 ftp 指令所在的機器，
     remote host 是 ftp 命令指定的機器）


     ☉ ascii   將傳輸模式設為 ascii 模式，通常用於傳送文字檔。此傳輸模式
                是進入 ftp 程式的初設值。


     ☉ bin (binary)    將傳輸模式設為二進位模式，通常用於傳送執行檔、壓縮
                        檔、圖形檔及影像檔等。


     ☉ dir     列出 remote host 上的檔案。


     ☉ ls      簡要列出 remote host 上的檔案。


     ☉ cd      更換 remote host 目前所在目錄。


     ☉ pwd     顯示 remote host 目前所在路徑名稱。


     ☉ get <檔名>      從 remote host 抓一個檔案到 local host 上(download)


     ☉ mget <檔名，可用萬用字元>       同上，可一次抓多檔。


     ☉ put <檔名>      從 local host 傳送一個檔案到 remote host 上(upload)


     ☉ mput <檔名，可用萬用字元>       同上，可一次傳多檔。


     ☉ lcd     改變 local host 目前所在目錄。有的 ftp 程式無此功能。


     ☉ ldir    列出 local host 上的目錄內容。有的 ftp 程式無此功能。


     ☉ mkdir   在 remote host 上建一個目錄。


     ☉ rmdir   在 remote host 上刪除一個目錄。


     ☉ delete/mdelete  刪除 remote host 上的檔案，mdelete 可刪多檔。


     ☉ open     建立與指定 remote host 的連線。進 ftp
                程式時若未指定 remote host，必須用此指令連上一個 remote
                host 才能作檔案傳輸。若用 close 指令結束與一個 remote host
                的連線後，可用此指令與另一個 remote host 連線。


     ☉ close   和目前已連線的 remote host 結束連線。


     ☉ bye/quit        結束連線並離開 ftp 程式。


     ☉ prompt [on|off]     若 prompt 為 on，在一次傳多檔時（mget/mput），
                ftp 會逐一詢問每個檔是否要做傳輸的工作，使用者必須逐一回答
                每個詢問。若 prompt 為 off，則不會詢問，每個檔皆做傳輸。系
                統預設值為 prompt on。


     ☉ hash    若為 on，傳輸時每傳 8K（隨設定而異），就會印出一個 "#"


     ☉ user    登錄時，若 username 或 password 打錯，可用此命令重新登錄，
                登錄成功後才能開始傳檔。


     ☉ help    線上輔助指令。
  □ anonymous ftp


     ☉ 有不少 ftp 站存放不少有用的資料並開放給各地的使用者抓取。這些 ftp
        站均提供了一個公用帳號 anonymous (或 ftp），使各地使用者能利用此帳
        號抓到該 ftp 站上有用的檔案。anonymous ftp 站在詢問 password 時會
        要求使用者輸入其 E-mail address，為尊重該 ftp 站管理者，請使用者遵
        照其使用規則，輸入完整 E-mail address，尤其是外國的 ftp 站，以維護
        台灣的形象。


     ☉ 目前國內最大 ftp 站是校園網路策進會提供的 NCTUCCCA.edu.tw，台大的
        使用者可用 ftp.ntu.edu.tw，兩者的內容一樣。該站的規模在亞太地區是
        數一數二的，定時會從國外各大站傳送（Mirror）檔案回來，使國內使用者
        可在該站以較快的速度抓所需的檔案。由於連往國外線路負荷很重且速度很
        慢，請確定你在 ftp.ntu.edu.tw 抓不到所要資料後才到國外抓。ftp.ntu.
        edu.tw 的 /PC/garbo 和 /PC/simtel 內有不少 for PC 的有用程式，值得
        參考。



  □ 如果你在連上網路的 PC 上使用 ftp，就可從 PC download 或 upload 檔案到
     你帳號內，也可直接 ftp 到其他 ftp 站去抓檔。

○ 第二節 ftp 的設定檔 .netrc ○

  □ 各位如果常常利用 ftp 傳檔，一定會覺得每次要打名字和密碼會很麻煩。只要
     在自己的根目錄下放 .netrc 檔就可以免去麻煩。
  □ .netrc 檔案內容的格式：
        machine xxxxxx  login xxxxxx    password xxxxxxx
     例如：
        machine cctwin.ee.ntu.edu.tw login b82000    password a9Xe7qW
        machine ftp.ntu.edu.tw       login anonymous password b82000@
  □ 請注意，因為 .netrc 檔中有你的密碼存在，所以必須更改檔案屬性，除了你
     自己之外的任何人皆不能讀寫，以免密碼外洩。最好只存 anonymous ftp 站的
     密碼就好了。

○ 第三節 ncftp -- 更好用的 ftp ○

  □ 有的工作站裝有 ncftp，它是一個較好的 ftp 介面，多了些方便的功能。它在
     連上 ftp server 後，自動幫你鍵入 ftp 作為 username，和你的 E-mail
     address 作為密碼。對常用 anonymous ftp 的人很方便。還有，它可以記錄你
     上一次離開該 ftp 站時的工作目錄，下次你再進入同一個 ftp 站時，便會自
     動更換工作目錄到你上次離開時的目錄去。此外，它還多了兩個有用的指令：


     ☉ predir
        ncftp 會把你上次 dir 的結果存起來，下 predir 指令可列出上次 dir 的
        結果，不必向 remote ftp server 再傳送目錄資料，這在網路傳輸速率慢
        時能節省時間，而且有 more 分頁看的功能。


     ☉ page
        可直接顯示 remote host 上文字檔的內容，例如目錄下有 00INDEX.TXT 或
        READ.ME，則用 page 00INDEX.TXT 或 page READ.ME 就可以直接將內容顯
        示，省去將該檔案傳回後再看的時間。
