一般按鍵

按鍵 動作
ESC 退回上一層視窗或取消執行
Enter 確認執行
F1 windows 或各程式的說明 Help
F2 重新命名
F3 尋找檔案或資料夾
F5 重整視窗
Ctrl + A 全選
Ctrl + C 複製
Ctrl + V 貼上
Ctrl + X 剪下
Ctrl + Z 復原
ß Backspace 查上一層或刪除前一個資料
Shift + ßBackspace 查下一層

　

在 Windows 中使用快速鍵

按鍵 動作
F10 啟動程式中的功能表列
Alt + 
功能表畫底線字母		 執行功能表上的相對應指令
Ctrl + F4 關閉多重文件介面 (MDI) 程式內的現有視窗
Alt + F4 關閉現有視窗或結束程式
Ctrl + C 複製
Ctrl + X 剪下
Delete 刪除
F1 在選取的對話方塊項目上顯示 [說明]
Alt + 空白 顯示現有視窗的系統功能表
Shift + F10 顯示選取項目的快速功能表(滑鼠右鍵)
Ctrl + ESC 顯示 [開始] 功能表
Alt + HYPHEN (-) 顯示 MDI 程式的系統功能表
Ctrl + V 貼上
Alt + Tab ●切換至上一次使用的視窗
●切換至其它視窗，請按住 Alt 而重複按 Tab
Ctrl + Z 復原

　

中文輸入按鍵

按鍵 動作
Ctrl + Shift 輸入法切換
Ctrl + 空白 中英切換
Shift + 空白 全半形切換
中文輸入模式下
Ctrl + Alt + ，		 跳出標點及符號鍵盤
自設快速鍵 控制台／輸入法／快速鍵／切換至倉頡
例：設定 Ctrl + Alt + 1=倉頡輸入法

　

視窗按鍵及組合按鍵

視窗按鍵： 

按鍵 動作
視窗鍵 開始功能表
視窗鍵 + R 執行功能
視窗鍵 + M 將全部視窗最小化
Shift + 視窗鍵 + M 還原視窗最小化的動作。
視窗鍵 + F1 執行 Windows 的Help功能。
視窗鍵 + E 呼叫檔案總管
視窗鍵 + F 尋找檔案及資料夾
Ctrl + 視窗鍵 + F ＝開始功能表／尋找／電腦，尋找電腦
視窗鍵 + Tab 切換工作列上的己執行的程式項目循環，
執行按 Enter 或滑鼠
視窗鍵 + Break 顯示系統內容對話方塊
視窗鍵 + D 將全部視窗最小化或復原
　 　
Alt + Tab 切換不同程式視窗
Shift + Del 在檔案總管下執行直接刪除的功能，不經資源回收桶
Alt + Enter 切換 MS-DOS 視窗為全螢幕或縮回原狀
Alt + F4 結束各應用程式，包含 Windows 的關機動作哦!

在對話方塊中使用快速鍵

按鍵 動作
ESC 取消目前工作
Spacebar ●若目前由按鈕控制，請按一下按鈕
●若目前由核取方塊控制，請選取或清除核取方塊
●若目前由選項按鈕控制，請按一下選項
Alt + 畫底線字母 請按一下相對應的指令
Enter 請按一下選取的按鈕
Shift + Tab 在選項中向後移動
Ctrl + Shift + Tab 在標籤中向後移動
Tab 在選項中向前移動
Ctrl + Tab 在標籤中向前移動
Backspace 若在 [另存新檔] 或 [開啟] 對話方塊中選取某資料夾，則開啟上一層的資料夾。
F4 在 [另存新檔] 或 [開啟] 對話方塊中開啟 [儲存到] 或 [查詢]。
F5 重新整理 [另存新檔] 或 [開啟] 對話方塊

Ctrl + Shift 
拖曳檔案時		 建立快速鍵

Application 鍵 顯示項目的快速功能表
F5 重新整理視窗內容
F2 重新命名
Ctrl + A 全選
Alt + Enter 或 
Alt + 按兩下		 檢視項目的內容

　

使用我的電腦與 Windows 檔案總管的快速鍵

按鍵 動作
Shift 請按一下 [關閉] 按鈕
 (僅 [我的電腦])		 關閉選取的資料夾與其上層資料夾
Alt + ← 後移到前一個檢視
Alt + → 前移到前一個檢視
Backspace 檢視上一層資料夾

　

使用 Windows 檔案總管的快速鍵

按鍵 動作
← ●若目前選擇的項目是展開的則加以隱藏
●選取上層資料夾
Num Lock + (-) 隱藏選取的資料夾
→ ●若目前選擇的項目是隱藏的則加以展開
●選取第一個子資料夾
Num Lock + * 展開目前選擇項目下的所有資料夾
Num Lock + (+) 展開選取的資料夾
F6 在左右窗格中切換
