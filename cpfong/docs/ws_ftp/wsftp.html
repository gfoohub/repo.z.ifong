◎WS_FTP的使用說明◎
WS_FTP95LE說明與安裝
FTP為個人電腦PC端與網路伺服器SERVER端檔案傳輸方式，一般最簡便與最常使用的就是Windows作業系統下之WS_FTPLE軟體。如果您還沒安裝，請按這裡..抓回去安裝。 抓回去後是ws_ftple.zip是壓縮檔，先利用WinZip解壓縮後即可進行安裝。

WS_FTP95LE使用

  1. 進入WS_FTP95LE：
    double click WS_FTP95LE的圖示即可開始執行。

  2. 登錄遠端Server帳號與連結：
    執行WS_FTP95LE之後，首先選擇欲進行傳輸的遠端伺服器，或是將這些伺服器資料一一填進各項設定中，說明如下：
    • Profile Name：填入伺服器名稱
    • HostName/Address：欲連往的伺服器ip位址
    • User ID：使用者名稱
    • Password：使用者密碼
    • Anonymous：選擇以Anonymous的id連接
    • Save Pwd：儲存使用者的密碼
    • 最後按下[確定]按鍵，進入伺服器。

  3. WS_FTP95LE檔案傳輸說明：
    若成功連接上伺服器後，會出現以下的畫面。要進行檔案傳輸時，首先將兩端目錄換到要進行傳輸的路徑，其次選擇本地端或是遠端要進行傳輸的檔案及傳輸模式，最後按下上傳或下傳按鍵，就開始進行檔案傳輸了，同時也會顯示檔案傳輸的進度。各項說明如下：
    • Local System：本地端資料，C:\temp為本地端目前所在路徑
    • Remote system：遠端資料，/ie3600為遠端目前所在徑
    • ChgDir：變更目錄
    • MkDir：建立目錄
    • 空白：填入欲尋找的檔案（須填全名）
    • View：以記事本開啟被選取的檔案
    • Exec：執行被選取的檔案
    • Rename：變更被選取的檔名
    • Delete：刪除被選取的
    • Refresh：重新顯示
    • DirInfo：所在目錄的相關資訊
    • ASCII：以ASCII的模式傳送資料
    • Binary：以ASCII的模式傳送資料
    • Auto：自動判斷模送的型式
    • ->：檔案上傳
    • <-：檔案下載
    • Close：結束連線
    • Connect：重新連接
    • Exit：結束WS_FTP95LE

    這樣子就能把個人電腦PC端檔案上傳至網路伺服器SERVER端，或將網路伺服器SERVER端下載個人電腦PC端
