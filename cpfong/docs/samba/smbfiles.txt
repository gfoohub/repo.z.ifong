Samba 的檔案

    * smbd： SMB server。 處理來自 client 的連結，做所有有關檔案、 權限、 使用者帳號的工作。
    * nmbd： NetBIOS name server。 幫助 client 找出 server 的位置， 做瀏覽的工作跟管理網域。
    * smbclient： Unix 主機用的 SMB client 程式。 
    * smbrun： 幫助 server 執行外部程式的小程式。
    * testprns： 測試 server 能否存取印表機的程式。
    * testparms： 測試 Samba 設定檔是否正確的程式。
    * smb.conf： Samba 設定檔。
    * smbprint： 讓 Unix 主機使用 smbclient 列印到 SMB server 的 script 檔。
    * 說明文件： 不要忘了讀一讀，會節省很多時間。
