Q. 如何安裝及重新安裝?
A. 用 root 和 port 安, 詳細過程, 重安時把 /var/log/news (note: log will default on /usr/local/news/log v2.2.3.0) 給刪了
Tip: 讓安裝者 (cpfong) 成為 news 的 group 會方便許多
install inn-2.3.3 short note


Q. 如何先弄成功一個只有 local group 沒有任何外面 server 的 news server ?
A. 按我有原文可以參考
  1. 把 Server 啟動再說 -- up and running
    1. /etc/syslog.conf 先弄好 file 別忘了 reload syslog
    2. cd /usr/local/news/db/; ../bin/makedbz -i -o; chown news history* more
    3. 用 default 的 newsfeeds
    4. 用 default 的 incoming.conf
    5. 最簡的 (也是內定的) active
    6. /usr/local/news/bin/inncheck 來檢查你的各種設定檔 我的 error
    7. run server (/usr/local/etc/innd.sh start) Error -- Sol
    8. 再 run 一次, 看看 log news.err
    9. ls /usr/local/news/run run

  2. 加一個新的群組 -- after running
    1. 它山之石
    2. 新的 group

  3. 文章存入的方式 -- I use cnfs
    1. vi ~news/etc/storage.conf
    2. down innd; rerun innd error
    3. vi ~news/etc/cycbuff.conf.ORG
    4. create a directory for cnfs
    5. cycbuff.conf.NEW
    6. shutdown and up once

  4. 設定網域或主機的讀寫權限
    1. vi ~news/etc/readers.conf

  5. 使用 news client 軟體連線上 localhost 來 post
    1. cd /usr/ports/chinese/tin/; make install
    2. tin -g localhost

  6. 到此應該在 localhost 上可以 post 文章了, 但其它的機器應不能在此 post
