目的：部署網頁

使用軟體:
1 github: 檔案所在空間
2 netlify: 控制台

帳號:
github, netlify 皆為共享軟體公司，所以都需要帳號來登入
github: 使用電子郵箱來登入
netlify: 用 github 帳號來登入，所以若 github 已登入則 netlify 也會自動進去, 所以如果 netlify 想登出則應 github 先登出

部署注意事項：
-- 1 由 wowchemy (在網址部份把你自己的名字改成 wowchemy 即會進入) 先把樣本 copy 到自己的 github
2 由 netlify 來部署
3 所有的部署成功或訊息要在 netlify 上查，因為 github 只是空間

更動:
1 下載 zip (在 github code 那兒) 到你的機器
2 在機器上使用 mobaxterm 來 grep 內容物以取得要改的檔案

有關網域名相關問題可以查 ./20210504-relation.txt 文件
