動作

利用 控制台/程式與功能 開啟 Windows 功能之 telnet (客戶端) 功能 。執行 CMD 進入命令視窗模式，執行 telnet 192.168.240.200，輸入帳號 tXX、預設密碼 passwdXX (注意：密碼打時主機不回應) 執行 passwd 改密碼，先輸入原密碼 passwdXX ，再輸入兩次指定之密碼 ， 需出現 successful 表示更改密碼成功。
