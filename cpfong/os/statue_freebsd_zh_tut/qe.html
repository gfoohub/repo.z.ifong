10.10. qe - qe 是一個模仿 PE2 的編輯程式

qe 是一個模仿 PE2 的編輯程式，叫 qe 的原因只是因為字母 Q 是排在 P 之後。和 DOS 不同的是，UNIX 沒有那麼多鍵可用，而且不同的 Terminal 的鍵碼也略有不同。因此一些常用的 function 最好定義到 Control Key 或 Meta Key 上，以免不堪使用。

安裝 chinese/qe。
