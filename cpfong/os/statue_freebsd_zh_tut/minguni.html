6.14. mingunittf - 香港補增字符集2001

mingunittf 包含了香港補增字符集2001的所有字。

mingunittf 的安裝:

# cd /usr/ports/outta-port/mingunittf
# make install clean

由於搭配 ttfm 的 xttfm 模組，因此在 XF86Config 裡面一定要 Load "xtt" 才行。
