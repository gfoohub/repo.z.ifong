13.24. gaim-openq - Gaim QQ client

騰訊 QQ 是大陸非常流行的一套即時傳訊系統(Instant Messenger, IM)， 目前可以安裝的版本有 perl-oicq，以 Java 寫的 LumaQQ， 以及以 gaim plugin 製作的 gaim-openq。

安裝 net/gaim-openq。

WWW: http://openq.linuxsir.org/
