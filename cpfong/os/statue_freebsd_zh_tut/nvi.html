10.9. nvi - 類似 vi/ex，有多種語言修補，預設為 big5

有 nvi-big5、 nvi-enc-cn、 nvi-enc-tw 等不同的套件。 BIG5 或 GB 相容的 vi 操作介面編輯器 vi 是 UNIX 的標準編輯器，此程式和中文繁體、enc-cn、 euc-tw 相容。

安裝 chinese/nvi-big5。

編輯 ~/.nexrc

set noskipdisplay
set displayencoding=big5
set inputencoding=big5
set fileencoding=big5
set autodetect=tw
