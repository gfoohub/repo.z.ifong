7.9. SCIM - Smart Common Input Method platform

SCIM 的全名是 Smart Common Input Method，是在 freedesktop 組織下的一個發展計畫， 乎合 iiimf (Internet Internet Input Framwork) 的架構， SCIM 提供的不只是二十多種的各國語言的輸入法， 可以使用一個豐富的介設定面。

目前仍然缺乏繁體輸入法表格。

textproc/scim 本身並不包含輸入法， 只有提供一個輸入法平台，你必須額外安裝 chinese/scim-chinese 或是 chinese/scim-tables 來安裝輸入法。

安裝完後，必須設定 XMODIFIERS 的環境變數：

# setenv XMODIFIERS "@im=SCIM"

目前仍然缺乏繁體輸入法表格。kanru 利用 libchewing 建立了 scim-chewing 專案，目前正在發展中。
