13.26. LumaQQ - Java QQ client

這是 bbs.smth.org 上的 luma 使用 Java 語言開發的 QQ 客戶端， 界面基於 swt，它的一大好處就是和 windows 平台上的騰訊 QQ 客戶端非常相像。 要使用它，必須先安裝 java/jdk14， 這裡可以下載已經編譯好的 package。首先確認正確設置了 JAVA_HOME 環境變數，解壓縮這個 package，運行其中的 lumaqq 腳本就可以使用了。 如果想自己編譯，需要安裝 java/eclipse ，因為 LumaQQ 需要其中的 swt，而目前 swt 並沒有做成一個單獨的 Port。 然後通過 cvs 取出 LumaQQ 的代碼，並使用 devel/ant 編譯。
