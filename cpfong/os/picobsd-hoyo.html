安裝一

mkdir /usr/src/release/picobsd/test
cp -R bridge test
cd /usr/src/release/picobsd/test
/usr/src/release/picobsd/build/picobsd

錯誤一： crunchgen: Can't execute: cd /usr/ports/picobsd/ssh-picobsd/work/ssh-1.2.27 && echo -n /bin/pwd

方法一：不使用 ssh，將 crunch.conf 以下程式加上 # 註解即可

progs sshd
special sshd objvar SSHD_OBJS
special sshd srcdir /usr/ports/picobsd/ssh-picobsd/word/ssh-1.2.27
special sshd objdir /usr/ports/picobsd/ssh-picobsd/word/ssh-1.2.27
ln sshd ssh
ln sshd ssh1
ln sshd scp

方法二：將 ssh 的套件補上

cd /usr/ports/security/openssh

make install

安裝二

 

安裝：製作軟磁片

Windows：請到這裡下載 fdimage.exe
fdimage picobsd.bin a:

UNIX：使用 dd 即可
dd if=picobsd.bin of=/dev/fd0
