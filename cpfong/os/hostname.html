TCP/IP Address + Router or Gateway + 主機名稱 + DNS IP + Domain 設定

Last Update : 21 Apr 2004

例子 : -
Server Name = haha123
Domain Name = coxford.net
TCP/IP address = 201.65.7.2
Router IP = 201.65.7.1
IP Range = 201.65.7.0 至 201.65.7.255
DNS IP = 202.134.105.130, 202.134.105.135

主機名稱設定

把主機名稱 haha123 放入 /etc/nodename

把主機名稱 haha123 放入 /etc/hostname.???0 的網絡介面卡
( 此例為 elxl0 是網卡的名稱，/etc/hostname.elxl0，在安裝 Solaris 時，只要你選擇網絡連線，這個檔案便會自動產生。)

TCP/IP address 設定

把 201.65.7.2 　haha123 放入 /etc/inet/hosts

把 201.65.7.0 　255.255.255.0 放入 /etc/inet/netmarks

Router IP 或 Gateway 設定

把 201.65.7.1 放入 /etc/defaultrouter

Domain Name 設定

把 coxford.net 放入 /etc/defaultdomain

DNS IP 設定

把
search haha123
nameserver 202.134.105.130
nameserver 202.134.105.135
放入 /etc/resolv.conf

更改 /etc/nsswitch.conf 的設定

把 hosts: files 改為 hosts: files dns

或

# cd /etc
# mv nsswitch.conf nsswitch.conf.orig
# cp nsswitch.dns nsswitch.conf
