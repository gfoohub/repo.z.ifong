如何去找相關的資料
  • MySQL 站台@台灣
  • 搜尋
    第四章 新建資料庫
    1. 簽入系統:
      1. ssh yourname@cpfong.fjtc.edu.tw
      2. mysql -u yourname -p -h localhost 註: u為user, p為password, h為host

    2. SQL 語法:
      1. SELECT select_list FROM table_list WHERE condition 大寫為關鍵字, 小寫為自選
      2. BUY thing = price AS {cash | check} [, thing_name= ] {} 一定, | 或, [] 選擇

    3. 顯示資料庫 ( SHOW DATABASES )
      1. SHOW DATABASES [LIKE wild]

    4. 使用資料庫 ( USE database )
      1. USE db1;

    5. 新建表格 (CREATE TABLE table_name) or CREATE
      1. CREATE TABLE table_name (column_name datatype [NULL|NOT NULL] [,column_name datatype [NULL|NOT NULL]]...)
      2. create table authors (au_id char(11) not null, state char(2) null);
