和 Mysql 的伺服器溝通的方式:
	1. 在系統提示符號下打: mysql -p 以進入 mysql> 的系統提示符號準備溝通,
	接下來你應都打 mysql 看得懂的指令, 例如: show databases;
		優點: 一步一步來可以每次有問題即立即更正
		缺點: 慢, 因為每個動作都要寫, 且不能把所有步驟記錄下來

	2. 把程式先行使用 vi 寫下來, 接下來在系統提示符號下一次完成 mysql -p < ./file.sql
		優點: 快, 一次完成
		缺點: 偵錯不易
