課程簡介

在 UNIX 系統上使用 perl 撰寫程式以為下學期撰寫網路程式之先前準備.介紹 scalar, array, subroutine, hash, 基本輸入/輸出, 正規表示式, 控制結構, 檔案代號, 目錄, 行程, sorting, DBM. 使用 telnet 為工具,以登入 vnc.mis.fjtc.edu.tw 主機, 在主機上進行 perl 程式之撰寫. Perl 語言在 Web 程式設計的發展過程中，佔有舉足輕重的地位，尤其是在 ASP、PHP 尚未興起之前，所有使用者與伺服器的溝通，完全是靠 CGI（Common Gateway Interface）來達成，而這些 CGI 背後的程式語言，至少有一半以上是靠 Perl 來完成使命的。 Perl 是一種底稿式語言（Scripting Language），因此和一般其他底稿式語言（如 MATLAB、JavaScript、VB、Tk/Tcl、Unix Shell）的特色類似，具有下列特色：資料型態定義較鬆散，如字串和數值可自動互換。直譯式語言，因此不需要編譯（Complie），但須依賴 Perl 直譯器（Intepreter）來執行程式碼。由於是直譯式語言，因此程式碼執行效率較差。由於不需要經過編譯程序，因此可以進行快速雛形設計（Rapid Prototyping）。
