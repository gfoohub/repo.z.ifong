第二章: 純量資料
    純量
  1. 純量 (scalar) 代表符號為 $
  2. 純量的意思為單數
  3. 和它相對的 @ 表陣列 (即複數)
  4. scalar 可以有數值及字串
    數值之 scalar
  1. 十進制 print 23
  2. 八進制 print O17
  3. 十六進制 print 0x1f
  4. 二進制 print 0b10
  5. 數字運算元 + - * / print 2+3
    字串
  1. 單引號及雙引號中間的全為字串,但有不同之意義
  2. 單引號表示 "不要" 解釋, 雙引號則表示要解釋
  3. ' 中間的任何字都表示它本身, 但有一個例外為單引號
  4. print 'x % $ ^ & *'; print 'x\a'; print 'X\'mas';
  5. " 中間若遇到 \ 則有特別意義 (見課本 p29)
  6. 字串運算元 . x print 'a' . 'b'; print 'a' x 2;
    數字和字串的混合使用
  1. 看你用的運算元是數字或文字的運算元, perl 會自動決定要如何計算
  2. print '1' + '2'; print '1' . '2';
  3. print 'z' . 5 * 7;
