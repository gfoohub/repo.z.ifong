heritage:
private (私人) -- protected (保護) -- public (公開)
  1. private: 只有經由 method 才可以取得 data
  2. protected: 同一 package 才可以
  3. public: 皆可
point0.java -- circle.java
  1. point 為最簡之 class
  2. testpoint 中有一個 main 來叫 point
  3. circle 可以自成一個 class
  4. circle 也可以 extends from point (但此時它使用的變數不可以是 point 中的 private 的)
