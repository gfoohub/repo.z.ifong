本課程主要運用 Java 程式語言介紹物件導向程式設計的基本觀念,並探討 Java 程式設計之相關技術與應用實務。基本程式設計觀念、物件導向程式設計、視窗GUI程式設計、集合物件的應用、跨平台的Java網路程式設計、資料結構程式設計。 bject-oriented programming，縮寫：OOP），指一種程式設計典範，同時也是一種程式開發的方法。它將物件作為程式的基本單元，將程式和資料封裝其中，以提高軟體的重用性、靈活性和擴充功能性。當我們提到物件導向的時候，它不僅指一種程式設計方法。它更多意義上是一種程式開發方式。在這一方面，我們必須了解更多關於物件導向系統分析和物件導向設計（Object Oriented Design，簡稱OOD）方面的知識。
