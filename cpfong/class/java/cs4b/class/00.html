1. 進入 CMOS 改成用 cd-rom 開機
  2. 放入光碟
  3. 出現 boot: 鍵入 root dma nolcd ttf2ram
  4. 在上頭有紅色鈕可關機
  5. 把行動碟放入
  6. mount /dev/sda1 來掛上
  7. umount /dev/sda1 才可以移除
  8. mount /dev/sda1 只要一次即可, 然後可以 ls /mnt/sda1 看內容
  9. cp /mnt/sda1/j2sdk-1_4_2_01-linux-i586.bin ~
  10. cd (到你簽入的地方, 也可以打 cd ~)
  11. ./j2sdk-1_4_2_01-linux-i586.bin
  12. export PATH=/home/root/j2sdk1.4.2_01:$PATH
  13. vi /mnt/sda1/welcome1.java
  14. 課本內容打入
  15. javac welcome1.java
