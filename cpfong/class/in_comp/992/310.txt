chap 7 使用作業系統

7.6 個人電腦的作業系統:
DOS (磁碟作業系統): 1980 年開始至今
	缺點: 只可單人用, 網路不內建, 只支援 16 bits 程式
	優點: 小速度快

Windows 9X: 由 Window 3.0 3.1 3.11 進化而來, Windows 9X
	指的是 Windows95  Windows98 WindowsMe
	優點: 可多工進行, 網路內建, 支援 32 bits 程式

Windows 2000 專業版: Windows 2000 主要給小型企業使用, 
	它增強了網路和安全部份, 個人較少在用, 

Windows XP: 發表於 2001 年 10 月, 主要強項: 數位媒體, 進階的網路通訊

Windows Vista: MS 公司於 2007 年推出的最新作業系統， 有
更輕鬆, 更多樂趣, 更具行動力, 更安全, 但是相對的對於硬體
的要求也更高， 所以價格也居高不下， MS 公司計劃自 2010 年
開始不支 XP， 如此會刺激 vista 的買氣

Macintosh : 蘋果電腦是出版商, 多媒體開發者繪圖者的最愛.
	桌上型電腦使用的 unix: Linux 是自由軟體 

桌上型電腦使用的 Linux: 因為原始檔的開放， 而造成各種不同版本 Linux
	的蓬勃發展， Linux 是全 32 位元， 支援多人和序多重處理的多工處理作業系統

除了 UNIX 和 window 二大陣營的作業系統外尚且有一稱為網路偏仵作業系統
網路作業系統為一種在網路伺服器上跑的作業系統

內嵌式作業系統: 把 OS 放入特定的裝置之中， 不一定要放入電腦中， 可以放到家電用品或是行
動電話 ...
