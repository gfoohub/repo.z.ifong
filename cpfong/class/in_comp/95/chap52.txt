5.2

電腦的二個主要資料處理單元: CPU 記憶體

CPU 的二個基本元件: CU (控制單元) ALU (算術邏輯單元)
	CU: 以 microcoe 的方式 來執行 CPU 的指令
	ALU: 執行 + - * / 和 > == < 等算術邏輯運算

CPU 執行一個指令的完整週期為機器週期
	機器週期: 指令週期 (提取, 解碼)  + 執行週期 (執行, 儲存)

CPU 的效能的計算單元: MIPS (millions of instructation per second)

MultiThread (多緒) 為一個 cpu 可以同時執行多個工作


記憶體: 所有程式和資料要執行前要放入記憶體中. 所以它的大小會影響到
執行的速度, 內建的記憶體可以分成永久性和非永久性的.  

永久性記憶體: 個人電腦的 ROM (Read Only Memory) 中有 BIOS (basic 
input output system) 可以啟動電腦

非久性記憶體: 
	flash memory 可以用在 數位相機, usb 行動碟, mp3 播放器
	ram 則是在個人電腦的主機板上可以擴充的, 有 simm (單排)
	和 dimm (雙排) 的, nb 上則有小型的 dimm 

影響電腦處理速度的因素:
	暫存器, 記憶體, 內部時鐘, 匯流排, 快取記憶體
	暫存器: 位於 cpu 上的小且很快的記憶體
