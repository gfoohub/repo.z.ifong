第二章A: 網際網路介紹

何謂 internet: 網際網路為連接數以千計的全世界電腦在一起, 經由 internet 某一台電腦可以和世界上的任何電腦相通.

2.1 網際網路的歷史: 
 1.三十多年前 (1969) 美國國防部發為了實現網路可以在戰時某一部份被毀但仍可以運作的多條路徑, 於是把一些大學和國防電腦連線, 此個系統稱為 APRANET. 
 2. 1980 年國家基金會 (NSF) 接手這個計劃, 而 NSFnet 和 APRANET 連在一起即成了 Internet. 
 3. 1990 年商業型的 Internet 取代了學術型的網路.

2.2 網際網路的主要服務
 1. 全球資訊網 www
 2. 電子郵件 email
 3. 新聞 news
 4. 檔案傳輸 ftp
 5. 交談 chat
 6. 立即傳訊 msn

2.3 www
 1. 起源於瑞士日內瓦的實驗室, 起初是為了創造一個鏈結以連接到不同的電腦存取資料
 2. web 使用 http 的通訊協定, 使用的語言為 html (使用 ie 的觀看原始檔即可以看到)
 3. http 使用的定址格式為 url 一個 url 看起來像 type://address/path 前面的 type
    為傳輸型態, address 則為網路上的一個位址 (主機), 最後的 path 則為一個檔案
    其中的 address 以點串起數個欄位, 例如: www.hinet.net 
    edu 表教育, com 表商業, gov 表政府, net 表網路, org 表組織, idve 表個人 
 4. 目前的瀏覽器透過 plug-in (內嵌式程式) 可以看到各種影音的檔案, 包括串流音訊
    及串流視訊 (streaming video)

2.5 搜尋網站
 1. 分類目錄 (雅虎) vs 搜尋引擎 (google)
    分類目錄: 經由主動登錄和人為篩選方式，蒐集數以萬計的網站資料，整理成主題式的分類目錄
    搜尋引擎: 以網頁為蒐集對象，提供關鍵字查詢服務
 2. http://www.google.com.tw
 3. http://www.yahoo.com.tw
 4. 印表機, 雷射印表機, hp 雷射印表機 
 5. 使用進階搜尋
 6. 使用超搜引擎 (metasearch engine): 超搜引擎會到各個搜尋引擎幫你找
