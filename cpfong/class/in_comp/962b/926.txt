第一章B: 電腦系統內部探索

p26 1.5 電腦系統元件
完整的電腦系統要包括四個部份: 硬體, 軟體, 資料, 使用者

硬體: 電腦上實際可以觸及的部份, 例如: 主機板, 鍵盤 ...
軟體: 使電腦可以執行的一連串命令, 程式... 等, 軟體是觸摸不到的.  
資料: 個別的資訊組成資料, 經由軟體可以把資料開啟, 資料也是觸摸不到的
使用者: 操作電腦的人.

p27 1.6 資訊處理的循環
資訊處理的循環共有四個步驟: 輸入, 處理, 輸出, 儲存

電腦硬體的主要裝置, 共四大類: 處理器, 記憶體, 輸出入, 儲存器
以下分別介紹:

處理器: 即中央處理器, 為電腦的心臟, 又名 cpu 為電腦中最主要的處理硬體,
位於主機板上, 形狀大都為正方形, 因它的運作時間長, 容易產生熱, 所以上方
通常有散熱片和風扇.

記憶體: 暫時存放或永久資料的地方, 二種最重要的記憶體為:
ram 和 rom 通常它們的容量都不大

ram 隨機存取記憶體可以暫時存放資料, 電腦電源關閉後資料
即不存在, 所有的程式和軟體要執行前要先放到 ram, 目前賣出的電腦
所配的 ram 都超過 1G 以上.

rom 唯讀記憶體可以永久的保存資料, 其容量因此較小, 電腦
開機時會使用到 rom 以認知電腦的裝置有那些. 

輸出入裝置: 鍵盤, 滑鼠, 掃描器, 數位相機, 螢幕, 印表機
...  等可以和電腦溝通的設備我們稱為輸出入裝置.

儲存器: 可以永久把資料存起來的地方, 最普遍的硬體為硬碟
另外有隨身碟和 mp3 都為常見的儲存器, 光學的儲存器則有
cd, dvd 儲存器和記憶體都可以存資料, 但儲存器可以存許多
的資料, 大小都是 ram 的數千到萬倍 (即數百 g 到 T)

軟體部份則有三種基本的形式: 作業系統, 應用軟體和資料
一些著名的作業系統: Window XP, Window Vista, Linux, FreeBSD,
Mac, Sun, Novell

一些著名的應用軟體: Word, Excel, Photoshop, Outlook, 小畫家,

資料: 作業一, 旅遊相片一, ...
