的服務上第一個使用的

膫解電子郵件: 想要使用電子郵件, 你必需有一支電子郵件的程式
(Outlook) 或者到網路上提供郵件服務的公司 (gmail, yahoo) 註冊

電子郵件看起來會像:
jsmith@aol.com

所謂的郵件伺服器 (mail server) 是一台電腦主機, 這台主機 24 
小時開機, 可以接收你的郵件.

Outlook 程式: 何謂 收件者, 副本, 主旨, 內容, 附加文件 ?
使用 msimn (Micro Soft Internet Mail News) 可以叫出 outlook

ftp 主機 5. 如果簽入失敗則 quit 重來前面動作乙次 6. bin
7 ls 8. get (put) 檔名

第三章A: 與你的電腦互動

3.1 鍵盤
何謂修飾鍵: Shift, Alt, Ctrl 三個鍵為修飾鍵, 它的功能為配合其它的鍵一同使用以產生不同的作用
1996 年開始, 所有 ibm 相容的鍵盤中多了二個額外用途的鍵: 開始鍵, 快速鍵
電腦如何接受鍵盤的輸入:  按下的鍵 -> 鍵盤控制器 -> buffer --> 系統軟體 -> cpu
介面: ps/2 usb 無線 


3.2 滑鼠
介面: ps/2 usb 無線
型式: 機械式, 光學式 

一般按鍵
F2		重新命名
F3		尋找檔案或資料夾
F5		重整視窗
Ctrl + A	全選
Ctrl + C 	複製
Ctrl + V 	貼上
Ctrl + X 	剪下

在 Windows 中使用快速鍵
Alt + TAB	在目前開啟的視窗中遊走
Alt + F4	關閉目前的視窗
Ctrl + F4 	關閉多重文件介面 (MDI) 程式內的現有視窗

中文輸入按鍵
Ctrl + Shift	輸入法切換
Ctrl + 空白 	中英切換
Shift + 空白 	全半形切換
Ctrl + Alt + ， 跳出標點及符號鍵盤 (中文輸入模式下)

視窗按鍵
視窗鍵 + R 	執行功能
視窗鍵 + M 	將全部視窗最小化
視窗鍵 + E 	呼叫檔案總管
視窗鍵 + F 	尋找檔案及資料夾
